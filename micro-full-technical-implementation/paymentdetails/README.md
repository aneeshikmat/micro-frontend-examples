# ال PaymentDetails Micro

## الوصف
يعرض ال PaymentDetails طرق الدفع المتاحة للمستخدم وسجل معاملات الدفع. يتيح للمستخدم إضافة سجلات دفع جديدة ويرسل إشعارات عن هذه الإضافات إلى ال micros الأخرى.

## الوظائف الرئيسية
- عرض طرق الدفع المتاحة للمستخدم
- عرض سجل معاملات الدفع
- إمكانية إضافة سجلات دفع جديدة
- إرسال إشعارات عن إضافة سجلات دفع جديدة

## ال Events
### يرسل:
- `payment:history:added`: يرسل هذا الحدث عندما يتم إضافة سجل دفع جديد.

## كيفية التشغيل
```bash
cd paymentdetails
npm install
npm run dev
```

يعمل هذا ال micro على ال Port 8103.
